Server repo/folder for json to typescript project

Built with Typescript. 

Ensure to create a '.env' file in the server folder

This env file is expected to contain the following variables
- OPEN_AI_API_KEY
- PORT (default: 4000)

To run,
first run 
```bash
npm install
```

then 

```bash
npm run dev
```

Using Yarn 

```bash
yarn dev
```
